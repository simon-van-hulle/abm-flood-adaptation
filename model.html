class model.model.AdaptationModel(*args: Any, **kwargs: Any)[source]

Bases: Model

The main model running the simulation. Extention of the mesa.Model class. It sets up the network of household agents, simulates their behavior, and collects data. The network type can be adjusted based on study requirements.

Parameters:

  • seed (int) – seed for the random number generator

  • wizard (Wizard) – wizard object containing all magic numbers

  • adaptation_cost (float) – cost of full adaptation to a flood

  • information_abundance (float) – abundance of information available to the agents (between 0 and 1)

  • steps_with_flood (list[int]) – list of steps at which a flood occurs

  • government_adaptation_strategies (list[str]) – list of adaptation strategies the government can use

  • information_cost (float) – cost of information - full information abundance per step

  • years_per_step (float) – number of years per step

  • subsidy_policy (callable) – function that returns the subsidy for a given household

  • max_initial_savings (int) – maximum initial savings

  • house_vs_savings (int) – ratio of houses to savings

  • avg_std_savings_per_step_vs_house (list[float]) – average and standard deviation of savings per step per house

  • avg_std_trustworthiness (list[float]) – average and standard deviation of trustworthiness

  • avg_std_trustworthiness_governnment (list[float]) – average and standard deviation of government trustworthiness

  • min_max_damage_estimation_factor (list[float]) – minimum and maximum damage estimation factor

  • min_max_rationality (list[float]) – minimum and maximum rationality

  • min_max_initial_risk_aversion (list[float]) – minimum and maximum initial risk aversion

  • min_risk_aversion (float) – minimum risk aversion

  • min_max_actual_depth_factor (list[float]) – minimum and maximum actual depth factor

  • avg_std_flood_influence_risk_aversion (list[float]) – average and standard deviation of flood influence on risk aversion

  • number_of_households (int) – number of household agents

  • flood_map_choice (str) – flood map choice. Can currently be “harvey”, “100yr”, or “500yr”.

  • network (str) – network type. Can currently be “erdos_renyi”, “barabasi_albert”, “watts_strogatz”, or “no_network”

  • probability_of_network_connection (float) – likeliness of edge being created between two nodes

  • number_of_edges (int) – number of edges for BA network

  • number_of_nearest_neighbours (int) – number of nearest neighbours for WS social network

  • grid (Network) – grid object for the network representation

  • G (nx.Graph) – graph object for the network representation

  • schedule (RandomActivation) – schedule for activating agents

step()[source]

introducing a shock: at time step 5, there will be a global flooding. This will result in actual flood depth. Here, we assume it is a random number between 0.5 and 1.2 of the estimated flood depth. In your model, you can replace this with a more sound procedure (e.g., you can devide the floop map into zones and assume local flooding instead of global flooding).
